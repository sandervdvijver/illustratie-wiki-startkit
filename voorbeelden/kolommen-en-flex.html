Kolommen, of how I learned to stop worrying and love flexbox

Hoe zet je beelden naast elkaar? Een van de veel voorkomende quandaries van de voorbije les(sen). Deze pagina maakt je wegwijs.

Druk op de d toets op je toetsenbord om debug mode te activeren. Zo zie je hoe deze pagina gestructureerd is!

Kolommen met flex

Vanaf je dingen naast elkaar wil zetten, loert flex om de hoek. Flexbox is een heel ding in CSS en hoewel het véél kan is het best een pittig baasje.

We proberen het concreet te maken met het voorbeeld van de kolommen. Alle elementen die je als 'kolom' naast elkaar wil zetten, dien je in je HTML te 'bundelen' onder 1 mama-element of container. Druk op d om te kijken hoe dat hier gebeurd is.

Aan dit container-element moeten we vervolgens display:flex toekennen om er een flexbox van te maken. Bijvoorbeeld met een class als:

.flex {
  display: flex;
  gap: 20px; /* de ruimte tussen de kolommen */
}

Je merkt in dit voorbeeld dat de paragraaf de afbeelding sterk doet verkleinen. Flexbox gaat nogal 'flexibel' om met de afmetingen die het toekent. Dit voelt best onintuïtief aan en passen we dan ook doorgaans aan. Zie hieronder.

Even brede kolommen met flex:1

Om alle kolommen in je flexbox dezelfde breedte te geven moet je de eigenschap flex:1 toekennen aan alle 'kinderen' van de flexbox.

Je kan dit ook in een eigen 'class' schrijven die je vervolgens in je html toevoegt aan je elementen:

.even-brede-kolommen > * {
  flex: 1;
}

Deze class zegt concreet: elke 'directe afstammeling' (> *) van een element met de .flex class, krijgt flex:1 toegekend en gaat daardoor even veel ruimte 'claimen'.

Centreer de afbeelding

Er wordt wel eens gelachen met het feit dat een van de moeilijkste dingen in CSS iets horizontaal en verticaal centreren is. Hoewel het al beter is dan vroeger, is dit toch minder intuïtief dan je zou denken.

Een nogal waterdichte methode om dit te doen op een 'kind' van je flexbox is de volgende eigenschappen toekennen via een class:

.centreer {
  display: flex;
  align-items: center;
  justify-content: center;
}

In feite maak je van dit element op zijn beurt een flexbox en zeg je dat alle items erin horizontaal en verticaal gecentreerd moeten worden. Een beetje overkill voor de puristen, maar het meest 'solide' van alle manieren.

Andere uitlijningen

Zo kan je ook elementen onderaan uitlijnen met volgende CSS-eigenschappen:

.align-onder {
  display: flex;
  align-items: flex-end;
  justify-content: center;
}

Of rechtsboven:

.align-rechtsboven {
  display: flex;
  align-items: flex-end;
  justify-content: center;
}

Hier moet nog wat vultekst komen zodat duidelijk zichtbaar is dat de afbeelding onderaan staat en niet in het midden. Dus nog wat tekst om dit echt te nailen.

Een afbeelding zijn kolom laten vullen

Bij afbeeldingen die groot genoeg zijn, gebeurt dit automatisch. Aangezien onze Guust dat niet is, moeten we hem een handje helpen.

Het is een goed idee om altijd, in elke stylesheet (dit noemen we ook wel een reset) volgende stijl te zetten:

img {
  max-width: 100%;
  height: auto;
}

Deze zegt: een afbeelding mag nooit breder (max-width) worden dan de volledige breedte van zijn container (100%).

Als je gevaarlijk wil leven, gebruik je i.p.v. max-width:100; width:100%;. Nu zal je afbeelding ALTIJD 100% van de breedte van zijn container innemen, ook al is hij er van nature misschien te klein voor.

Ik heb bovenstaande toegepast op de Guust hiernaast met een style="width:100%" attribuut op het img element.

Kolommen op kleinere schermen laten inklappen

Zoals de code er nu uitziet is alles dandy en wel op grote schermen, maar worden de kolommen wel heel krap op kleinere schermen. Om dit te voorkomen kan je met een media query werken.

Dit is een manier om aan je CSS te zeggen om andere stijlen toe te passen als de afmetingen van het beeldscherm kleiner (of groter) zijn dan een bepaalde waarde.

Een basic versie hiervan is:

@media (max-width: 820px) {
  .flex {
    flex-flow: column;
  }
}

Als je schermbreedte kleiner is dan 820px krijgt alles met de class flex flex-flow:column; voor de kiezen. Dat wil zeggen dat een flexbox vanaf dan verticaal gaat 'vloeien' ⬇ in plaats van het default horizontaal ⮕. Zo komen de items onderaan in plaats van naast elkaar te staan.

Ik heb dit al toegepast op alle flexboxen op deze pagina, dus deze zijn gsm-proofed.

Meer dan twee kolommen

Kan allemaal. Hoeveel elementen er directe kinderen van de flexbox zijn, bepaalt hoeveel kolommen deze gaat proberen maken.
